### ⚙️ Engineering Principles

- **Code Quality:** I favor clarity and correctness over cleverness. I avoid primitive obsession, preferring type-safe Enum comparisons, guard clauses, and flat logic structures for predictable execution.
- **Security First:** Application Security (AppSec) is mandatory. I enforce least-privilege access, strict input validation, and secure defaults to protect against vulnerabilities.
- **Pragmatic Development:** I consistently apply YAGNI—implementing only what current requirements demand to avoid over-engineering and speculative abstractions.
